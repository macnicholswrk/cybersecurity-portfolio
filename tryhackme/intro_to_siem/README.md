## Overview
This lab introduces **Security Information and Event Management (SIEM)** systems, their capabilities, and how they provide visibility into IT environments. The lab covers how incidents are detected, investigated, and escalated.

## Objectives
- Understand what a SIEM is and its role in cybersecurity
- Explore SIEM capabilities and dashboards
- Learn how alerts are generated and investigated
- Understand false positives, true positives, and rule tuning

## Skills Practiced
- Alert monitoring and investigation workflows
- Identifying key fields and event IDs in logs
- Understanding SIEM dashboards and their insights
- Evaluating alerts for true/false positives

## Technologies / Concepts
- SIEM dashboards
- Log correlation and analysis
- Incident investigation
- False positive / true positive alerts
